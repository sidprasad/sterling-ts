# How to run these demos


1. First run the forge server (e.g. `yarn run dev:forge`)

2. Then run `racket targetfile.frg -O sterling_port 17100`

3. Then open http://localhost:8081?17100 in a web browser. Eventually:
   1. The page will load
   2. You will then have to click run in the Explorer part of the page
   3. This should render the graph.
